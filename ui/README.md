# Cesium Collision Detector UI

A web-based 3D visualization application using Cesium and Node.js/Express.

## Setup

### Prerequisites
- Node.js 14+ installed
- npm or yarn package manager

### Installation

1. Install dependencies:
```bash
npm install
```

2. Build the application:
```bash
npm run build
```

3. Start the server:
```bash
npm start
```

The application will be available at `http://localhost:3000`

## Development

For development with hot reload:
```bash
npm run dev
```

This will start the webpack dev server at `http://localhost:8080`

## Project Structure

- `src/index.js` - Main application entry point with Cesium viewer initialization
- `index.html` - HTML template with Cesium container
- `webpack.config.js` - Webpack configuration for bundling
- `server.js` - Express server for production serving
- `package.json` - Project dependencies and scripts

## Features

- 3D globe visualization using Cesium
- Load and display CZML data
- Interactive toolbar with buttons
- Responsive design with dark theme

## Notes

- Ensure the `dist` folder is generated by running `npm run build` before starting the server
- Sample data files should be placed in the `SampleData` directory
- The application uses Cesium's workers and assets which are automatically copied during the build process
